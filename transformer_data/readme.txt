This folder contains one file.

------

123A.csv contains an example of electric usage data for one distribution transformer (with ID 123A).

Columns
1. datetime - date and time of interval being measured
2. power - electric usage in kW
3. power_factor - (constant) power factor rating for this dist. transformer
4. transformer_kVA - kVA rating of this transformer
5. transformer_kW - kW rating of this transformer (obtained from kVA)
6. utilitzation - percentage of transformers rated capacity which is being used during this interval
